# Tiny-Universe

大模型是当下深度学习领域的热点，但现有的大部分大模型教程只在与教给大家如何调用api完成大模型的应用，而很少有人能够从原理层面讲清楚模型结构、RAG、Agent以及Eval。所以本仓库会提供全部手写，不调用api的形式，完成大模型的RAG、Agent、Eval应用。
